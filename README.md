# Student Management REST API

A backend REST API built using Django and Django REST Framework to manage student records.

## Technologies Used
- Python
- Django
- Django REST Framework
- SQLite
- Git & GitHub

## Features
- Create student records
- Retrieve student details
- Update student information
- Delete student data
- Search students
- Pagination support

## API Endpoints

GET /students/
POST /students/
GET /students/{id}
PUT /students/{id}
DELETE /students/{id}

## Installation

1. Clone repository
git clone https://github.com/VaddeGaneshKumar/student-management-rest-api

2. Install dependencies
pip install -r requirements.txt

3. Run server
python manage.py runserver

Deployed using Render cloud platform.
